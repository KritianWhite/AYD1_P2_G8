|---------------------------------|--------------------------------------------------------------------|

| Caso de uso                    | CDU-008 Realizar reseña a la plataforma                            |

| Descripción                     | Se utiliza para que el cliente pueda realizar reseñas y calificar a la plataforma Huellita Feliz. |
